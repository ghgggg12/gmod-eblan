Это крутая локализация для игры Garry's mod для её установки нужно перейти по данным папкам:
GarrysMod\garrysmod\resource\localization\
И туда скинуть все файлы предварительно создав папку с любым названием. Выглядить должно вот так:
GarrysMod\garrysmod\resource\localization\"ваше название папки"\"файлы из репозитория"
Также вам нужно будет создать фото в папке localization для того чтобы язык отображался среди других языков.
Удачи!
